
# Lezhuan - 校园二手交易平台

*让校园闲置物品流转起来，让每一件物品都能继续发光发热*


## 项目介绍

Lezhuan（乐转）是一款专为高校学生打造的二手交易平台，基于 uni-app 跨平台框架开发。平台聚焦校园场景，解决学生闲置物品处理难、交易效率低、信任成本高等问题，通过简洁的界面设计和实用的功能，让二手交易更简单、更安全。

### 平台优势
- **校园专属**：仅面向在校学生，交易对象更可信
- **操作便捷**：3分钟完成商品发布，一键沟通卖家
- **多端覆盖**：支持微信小程序/H5，无需下载安装
- **功能完善**：从商品发布到交易完成的全流程支持


## 技术栈

| 技术类别 | 具体实现 | 说明 |
|---------|---------|------|
| 核心框架 | uni-app | 一套代码多端运行，降低开发成本 |
| 前端框架 | Vue 3 | 采用Composition API，代码组织更灵活 |
| 状态管理 | Pinia | 轻量级状态管理，替代Vuex |
| 样式解决方案 | CSS/SCSS | 支持变量和嵌套，便于样式复用 |
| UI组件库 | uni-ui | 专为uni-app设计的组件库，风格统一 |
| 开发工具 | HBuilderX | 提供uni-app专属开发环境和调试工具 |
| 版本控制 | Git + GitHub | 代码管理与协作开发 |


## 功能模块

### 1. 首页模块
- **个性化推荐**  
  根据用户浏览历史和校园热门程度智能推荐商品
- **分类导航**  
  10+常用分类（数码产品/图书教材/服饰鞋包/生活用品等）
- **轮播公告**  
  展示平台活动、新规通知和安全提示
- **快捷入口**  
  一键发布、我的订单、客服中心等高频功能直达
- **校园动态**  
  实时显示本校最新发布的商品和交易动态


### 2. 乐转模块（发布中心）
- **发布商品**  
  - 多图上传（支持拍照/相册选择，最多9张）  
  - 信息填写（标题/价格/新旧程度/交易方式）  
  - 分类与标签（提高曝光率，便于搜索）  
  - 预览功能（发布前查看效果）  
- **发布需求**  
  - 求购信息描述（物品名称/期望特征）  
  - 价格范围与有效期设置  
  - 匹配通知（有符合条件的商品时提醒）  
- **发布管理**  
  - 草稿箱（保存未完成的发布内容）  
  - 历史记录（查看过往发布记录）  


### 3. 分类浏览模块
- 多级分类筛选（一级分类→二级分类逐层筛选）
- 价格区间过滤（支持自定义价格范围）
- 排序功能（按发布时间/价格高低/热门程度）
- 关键词搜索（支持模糊搜索和搜索历史记录）
- 筛选条件记忆（保留上次筛选设置）


### 4. 商品详情模块
- **基础信息展示**  
  多图轮播、价格、新旧程度、发布时间、发布者信息
- **详细描述**  
  支持图文混排的商品介绍，包含使用情况说明
- **互动功能**  
  - 收藏/取消收藏（方便后续查看）  
  - 分享（微信好友/朋友圈/QQ等）  
  - 在线咨询（内置简易聊天功能，支持表情包）  
- **交易相关**  
  - 自提地点与时间说明  
  - 举报功能（举报违规商品）  


### 5. 个人中心模块
- **个人信息管理**  
  头像/昵称/简介设置，校园信息认证（提高可信度）
- **我的发布**  
  管理已发布的商品和需求（编辑/下架/重新发布）
- **我的收藏**  
  按时间排序的收藏列表，支持批量取消收藏
- **消息中心**  
  咨询消息/系统通知/交易提醒分类展示
- **系统设置**  
  账号安全/隐私设置/清除缓存/关于我们


### 6. 指南模块
- **新手指南**  
  图文教程：如何发布商品/如何联系卖家/如何交易
- **常见问题**  
  - 交易安全：如何避免被骗/纠纷处理方式  
  - 功能使用：如何修改信息/如何删除收藏  
  - 平台规则：发布规范/禁止交易物品  
- **意见反馈**  
  提交功能建议或问题报告，支持上传截图

## 安装与运行

### 前置条件
- 安装 [HBuilderX](https://www.dcloud.io/hbuilderx.html)（推荐最新版）
- 安装 Node.js（v14.0.0+，用于依赖管理）
- 微信小程序开发者工具（如需运行到微信小程序）


### 详细步骤

1. **克隆仓库到本地**
   ```bash
   git clone https://github.com/Ticnix/Lezhuan.git
   cd Lezhuan  # 进入项目目录
   ```

2. **安装项目依赖**
   ```bash
   npm install  # 安装package.json中声明的依赖
   ```

3. **在HBuilderX中运行**
   - 打开HBuilderX，点击「文件」→「导入」→「从本地目录导入」
   - 选择项目文件夹，等待加载完成
   - 运行到不同平台：
     - H5：「运行」→「运行到浏览器」→ 选择目标浏览器
     - 微信小程序：「运行」→「运行到小程序模拟器」→「微信开发者工具」
     - 其他平台：类似操作，选择对应模拟器


### 注意事项
- 首次运行微信小程序需配置开发者工具路径：  
  HBuilderX「设置」→「运行配置」→「微信小程序开发者工具路径」
- 微信开发者工具需开启"服务端口"：  
  「设置」→「安全设置」→ 开启服务端口
- 部分功能依赖后端接口，本地运行可能需要配置代理
